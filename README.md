# Mercado Negro D.C

Aplicación web tipo e-commerce desarrollada con React + Vite.

## Características

- Visualización de productos (moda y tecnología)
- Selección de colores con cambio dinámico de imagen
- Carrito de compras funcional
- Contador de productos
- Modo claro / oscuro
- Diseño responsive
- Interfaz moderna

## Tecnologías

- React
- Vite
- JavaScript (ES6+)
- CSS

## Instalación

```bash
npm install
npm run dev
